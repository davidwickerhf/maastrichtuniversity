Lecture notes §

Proof example §

Proof example

For all sets A, B, C, (B∩(Ac∪C)c=ϕ)⟺(A⊆Bc∪C)

Transclude of block

We try to prove that the statement is true (which we identified using Venn Diagrams)

First implication §

B∩A∩Cc=ϕ⟹A⊆Bc∪C

  • Let x∈A
    • Case 1: x∈C
      • ⟹x∈C∨x∈/B⟹x∈Bc∪C
    • Case 2: x∈/C
      • Assume x∈B⟹x∈A,x∈B∧x∈/C
      • x∈(B∩A∩Cc) → This is a contradiction as that set is empty
      • Therefore x∈/B → x∈Bx∨x∈C⟹x∈Bc∪C
Second implication §

A⊆Bc∪C⟹B∩A∩Cc=ϕ → We can either take the contrapositive, or take a contradiction

Proof by contradiction

  • We need to assume the set B∩A∩Cc is not empty → there is an element in that set: x∈B∩A∩Cc
    • then x∈B,x∈A and x∈/C
    • so (x∈Bc∨x∈C)∧(x∈B∨x∈/C) → Contradiction (the two sets are opposite)
    • so there is no x∈B∩A∩Cc → the set B∩A∩Cc is empty

Proof example (shorter) §

Proof example

For all sets A, B, C, (B∩(Ac∪C)c=ϕ)⟺(A⊆Bc∪C)

We prove: ¬(A⊆Bc∪C)⟺¬(B∩(Ac∪C)c=ϕ)

Subset: ∀x∈A:x∈B Not subset: ∃x∈A:x∈/B

  • ¬(A⊆(Bc∪C))
    • ⟺∃x∈A:x∈/(Bc∪C)
    • ⟺∃x∈A:x∈(Bc∪C)c (if an element is not in something, it is an element of the complement)
    • ⟺∃x∈A:x∈(B∩Cc)
    • ⟺∃x:x∈A∧x∈B∧x∈Cc
    • ⟺∃x:x∈(A∩B∩Cc) → X is in the intersection of the three sets
    • ⟺(A∩B∩Cc)=ϕ → The set is not empty

Power set §

Power sets

A power set of a set A is the set of all subsets of A

Example of a power set: A={1,2,3} P(A)={{1},{2},{3},{1,2},{1,3},{2,3},{},{1,2,3}} ∣P(A)∣=8 (set size is 8)

Example of a power set: B={i,ϕ} P(B)={ϕ,{1,ϕ},{1},{ϕ}}

Product sets

For two sets A and B, the product set A×B is defined as {(a,b) with a∈A∧b∈B}

  • (a,b) is an ordered pair
