## Habits Tracker
```dataviewjs
dv.span("**💊 Pills**")

const calendarData = {
    year: 2023,
    entries: []
}

for(let page of dv.pages('"daily notes"').where(p=>p.pills)){
    calendarData.entries.push({
        date: page.file.name,
        intensity: page.exercise
    })
       
}

renderHeatmapCalendar(this.container, calendarData)
```
